个人使用的一些 vim 轻量级插件

2016-1-16

  • vim-bbye 移除 buffer 同时尽可能保留原有的窗口，对多窗口编辑切换非常有帮助。

  • matchit.vim 扩展了 vim 的 %，对我来说主要是匹配 html 的 tag，以及 vim 的里面成对的关键词，例如 if endif， 据说新版 vim 自带该插件

  • qdo.vim 主要用于对 quickfix 搜索结果进行批量替换，例如：:Qdo %s/\<abc\>/ABC/gc | update

  • system.vim 整合了一些系统相关操作，包括 rm rename mkdir copy to clipboard

  • simple_pairs.vim 轻量的字符配对插件，很容易修改。 不是特别智能，但也不会像某些自作聪明的插件恶心你。 这个插件有些 bug， 而且用了 python 会导致大幅增加 vim 的初始化事件，我有一个用 vim 编写的修复版 simple_pairs.vim

  • complete.vim 简洁的关键词补全插件，只是让 tab 可以实现 <c-n> <c-p> 不自作聪明，不影响性能。

  • open.vim Mac 下快速打开当前行的 url 和 email，简单调用了下 open 命令

  • mycomment.vim一个快捷键搞定注释问题

  • vim-easygit 轻量级 git wrapper 插件，用以解决 fugitive难以扩展的问题

  • jscheck 只做 js 语法检查的 vim 插件，用以解决 eslint 之类插件速度太慢的问题

  • unite-extra 一些实用的 unite 源，包括 emoji command project 和 note

  • unite-git-log 整合 gitlog 到 unite 界面的插件

  • unite-js-func 快速定位 js 函数的 unite 插件

  • unite-location 整合 vim quickfix 和 location 列表到 unite 的插件

  • vim-iterm2-start 让 iterm2 执行任务，主要解决 vim 不能异步，执行耗时任务会卡住的问题。

  • vim-macos 包含些与 macos 交互的函数，包括 ItermOpen, 发送系统 keycodes 等

  • vim-run 快速执行当前文件并看到结果的小工具，主要用于学习新语言。

  • vim-v2ex 异步方式刷新 v2ex 的小插件

  • macdown.vim 使用 applescript 的快速预览 markdown 插件

  • todoapp.vim 使用 unite 的 todo 管理插件

  • unite-session 支持 restart 的 session 管理插件
